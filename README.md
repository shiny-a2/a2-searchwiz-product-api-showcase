# A2 Searchwiz Product API Showcase

Public-safe showcase for a private WooCommerce snapshot product API that supports search/product-feed consumers with rebuild and allowlist boundaries.

This repository is not a source mirror. Production source, endpoint URLs, allowlist data, catalog exports, and search integration details stay private.

## What This Demonstrates

- Snapshot-backed product API design for expensive WooCommerce data.
- Full and delta rebuild workflow thinking.
- Integration access boundaries and allowlist-style protection.
- Separation between snapshot generation, serving, and operational controls.
- Public-safe documentation of performance and integration tradeoffs.

## Tech Stack

- WordPress plugin architecture
- WooCommerce product data
- PHP
- REST API design
- Snapshot/rebuild pattern
