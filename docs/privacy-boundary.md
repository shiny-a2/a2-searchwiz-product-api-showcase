# Privacy Boundary

This public repository is designed for employer review and portfolio proof only.

## Public

- General snapshot API architecture.
- Public-safe notes on rebuilds, caching, integration boundaries, and performance.
- Sanitized samples that do not expose production contracts.

## Private

- Production source code.
- Endpoint URLs, allowlists, search provider details, product exports, logs, and credentials.
- Live catalog response shapes used by private consumers.

## Rule

Public samples may explain the pattern, but never expose live access rules or private product-feed contracts.
